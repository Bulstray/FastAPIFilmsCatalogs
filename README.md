## Develop

Check GitHub Actions after any push.

### Setup

Right click 'films-catalog' -> Mark Directory as -> Source root

### Configure pre-commit

### Install dependencies

Install all packages
```shell
uv sync
```

Install pre-commit hook:
```shell
pre-commit intall
```


### Install

Install packages:
```shell
uv install
```

### Run

Go to workdir:
```shell
cd films-catalog
```

Run dev server:
```shell
fastapi dev
```
